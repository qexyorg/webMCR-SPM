Установка - Шаг 3

Для правильной работы скриптов, перейдите в папку style/Default/
Откройте файл index.html и перенесите строчки
<script src="<?php View::URL('js/jquery.js') ?>"></script>
<script src="<?php View::URL('js/bootstrap.min.js') ?>"></script>
В верхнюю часть (перед <?php echo $content_js ?>)

Если вы используете версию 2.4+, то удалите строку
<script src="<?php View::URL('js/bootstrap-without-jquery.js') ?>"></script>
И, сразу перед <?php echo $content_js ?>, добавьте строки
<script src="<?php View::URL('modules/qexy/statics/js/jquery.js') ?>"></script>
<script src="<?php View::URL('modules/qexy/statics/js/bootstrap.min.js') ?>"></script>
