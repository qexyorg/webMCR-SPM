Название
- Название модуля статических страниц (Отображается в заголовке страницы и хлебных крошках)
Идентификатор
- Идентификатор, по которому будет доступна страница (Пример: /?mode=statics&do=page&op=ВАШ_ИДЕНТИФИКАТОР)
Статус
- Статус страницы
Уровень доступа к странице
- Минимальный уровень доступа к статической странице
Текст страницы
